AbsolventWeb

This app makes the work in graduation projects between the doctors and the students easier.

Code

Technologies:

  • React.js
  • React Hooks
  • Redux Toolkit
  • Router
  • SASS

Features:

  • Authentication & Authorization.
  • Doctor publishes graduation projects, Student browses projects and select one.
  • Create Workspace between doctors and students.
  • Create events (post, meeting, task), add comments, upload attachments.
  • Different Ui Admin, Student, Doctor.
  • Filter, search and pagination.
  • Export data by Admin to excel file.
